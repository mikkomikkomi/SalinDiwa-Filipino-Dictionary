# SalinDiwa - Mobile Filipino Dictionary

SalinDiwa is a mobile dictionary app designed to help users learn and translate Filipino words. It includes a search feature, a dictionary list, and a built-in quiz mode to test your vocabulary knowledge. **This was made, as an independent project, for a friend's research.**

## Features

* **Word List:** Browse through a collection of Filipino words complete with definitions and images.
* **Quiz Mode:** Test your knowledge with a built in mini game. You can choose to guess the word or guess the meaning.
* **Custom Themes:** Switch between light mode and dark mode. 
* **Offline Access:** All text and images are saved locally so you do not need an internet connection to use the app.

## Tech Stack

* React Native
* Expo
* TypeScript

## How to Run Locally

1. Clone this repository to your local machine.
2. Open the folder in your terminal and run `npm install` to get all the dependencies.
3. Run `npx expo start` to start the local development server.
4. Scan the QR code with the Expo Go app on your phone to test it.
